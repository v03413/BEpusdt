# 国际化语言文件

## 文件说明

- `zh.json` - 中文翻译
- `en.json` - 英文翻译

## 使用方式

页面会自动根据浏览器语言设置选择对应的语言包：
- 如果浏览器语言是中文（zh），则显示中文
- 如果浏览器语言是英文（en），则显示英文
- 其他语言默认显示英文

用户也可以通过右上角的语言切换器手动切换语言，选择会保存在 localStorage 中。

## 添加新语言

如果需要添加新语言，请：
1. 在此目录创建新的语言文件，如 `ja.json`（日语）
2. 复制 `en.json` 的结构，翻译所有内容
3. 在模板文件的语言切换器中添加新选项
4. 更新 JavaScript 代码中的语言检测逻辑

## 翻译内容

包含以下内容的翻译：
- 页面标题和说明
- 订单信息标签
- 倒计时文字
- 收款地址标签
- 转账说明
- 弹窗提示消息（支付超时、网络确认、支付成功）
- 复制功能反馈
