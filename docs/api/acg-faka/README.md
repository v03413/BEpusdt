# 异次元发卡 BEpusdt 对接指南

异次元应用商店提供的 `Epusdt` 插件存在一些问题，本文档提供优化适配版本的安装和配置方案。

## 对接流程

### 1. 前置条件

请先按照异次元官方文档完成 `Epusdt` 支付插件的标准安装流程。

- 位置：应用商店 → 支付接口
- 搜索并安装官方 `Epusdt` 插件

### 2. 插件更新

[下载兼容适配版本](https://raw.githubusercontent.com/v03413/bepusdt/refs/heads/main/docs/api/acg-faka/Epusdt.zip)

**安装步骤：**

1. 进入异次元网站根目录
2. 定位到 `app/Plugins/Epusdt` 目录
3. 删除目录内所有文件
4. 解压下载的插件包，将所有文件移入该目录

**权限配置：**

⚠️ 确保插件目录权限为 `www:www`（通常为默认配置）。如权限不符，请执行：

```bash
chown -R www:www app/Plugins/Epusdt
chmod -R 755 app/Plugins/Epusdt
```

### 3. 插件配置

在异次元后台进行以下配置：

| 配置项  | 说明                           | 示例                         |
|------|------------------------------|----------------------------|
| 支付网关 | 必须以 `http(s)://` 开头，以 `/` 结尾 | `https://api.example.com/` |
| 对接密钥 | 从 BEpusdt 后台获取               | -                          |
| 法币单位 | 按需选择，默认推荐 `CNY`              | `CNY` `USD` `JPY`          |

### 4. 创建支付方式

在异次元后台完成以下操作：

1. 导航到：支付接口 → 添加支付
2. 选择支付插件：`BEpusdt`
3. 根据业务需求填写相关参数
4. 保存配置

配置完成后，系统将自动启用该支付方式。
