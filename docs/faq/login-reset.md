# 后台登录信息重置

当忘记登录密码时,可通过 `reset` 命令重置后台管理入口及账号信息。

## 重置方法

### Docker 容器部署

1. 查询容器 ID:

```bash
docker ps -a
```

2. 执行重置命令:

```bash
docker exec -it <容器ID> bepusdt reset
```

3. 命令执行成功后,将输出新的登录凭证:

```
重置成功,对应信息如下:
管理员账号: 4bd57acf
管理员密码: 40f7db37
后台管理入口: /d57acf0610
请妥善保存以上信息!
```

### Linux 二进制部署

1. 定位二进制文件路径:

```bash
which bepusdt
```

2. 执行重置命令(假设二进制文件位于 `/usr/local/bin/bepusdt`):

```bash
/usr/local/bin/bepusdt reset
```

3. 命令执行成功后,将输出新的登录凭证:

```
重置成功,对应信息如下:
管理员账号: 4bd57acf
管理员密码: 40f7db37
后台管理入口: /d57acf0610
请妥善保存以上信息!
```

## 安全提示

⚠️ 请在重置后立即修改默认密码,并妥善保管新的登录信息。
